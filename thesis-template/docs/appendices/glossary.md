# Glossary

| Term | Definition |
|------|------------|
| Formics | The diploma project: a web platform for dynamic forms. |
| Template | A reusable form definition created by a user. |
| Question | A field inside a template. |
| Response | A submitted form instance. |
| Answer | A stored value for one question in one response. |
| Public form | A template available to guests. |
| Private form | A template available only through authenticated/authorized access. |
| Correct answer | A value configured by the author for automatic checking. |
| SPA | Single Page Application. |
| API | Application Programming Interface used by frontend to communicate with backend. |
| JWT | JSON Web Token used for stateless authentication. |
| ORM | Object-Relational Mapping tool. |
| OLTP | Transactional database workload focused on operational reads/writes. |
| SSE | Server-Sent Events, a one-way live update stream from backend to frontend. |
| CORS | Browser security mechanism controlling cross-origin requests. |
| Docker Compose | Tool for running multiple containers as one local stack. |

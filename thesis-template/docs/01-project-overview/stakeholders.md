# Stakeholders and Users

## Stakeholder Groups

| Stakeholder | Role in Project | Needs |
|-------------|-----------------|-------|
| Form author | Creates and manages templates | Fast form creation, response review, public/private control, analytics. |
| Respondent | Completes forms | Simple access to public forms, clear questions, reliable submission. |
| Administrator | Manages users and roles | Visibility into users, role management, secure access. |
| Diploma committee | Evaluates the project | Evidence of architecture, implementation quality, deployment, and documentation. |
| Supervisor | Guides scope and quality | Clear progress, realistic feature boundaries, honest limitations. |

## Personas

### Persona 1: Form Author

| Field | Description |
|-------|-------------|
| Role | Teacher, team lead, community organizer, or internal coordinator. |
| Tech level | Medium. Comfortable with web forms but not database administration. |
| Goal | Create a structured form, collect answers, and review results quickly. |
| Frustration | Manual checking and scattered response files take too much time. |
| Scenario | Creates a quiz-like form with correct answers, publishes it, and reviews analytics. |

### Persona 2: Respondent

| Field | Description |
|-------|-------------|
| Role | Student, team member, event participant, or guest user. |
| Tech level | Low to medium. Expects a simple browser form. |
| Goal | Fill in a form without registration when public access is allowed. |
| Frustration | Confusing navigation or login requirements for public forms. |
| Scenario | Opens the public form link, answers questions, and submits the form. |

### Persona 3: Administrator

| Field | Description |
|-------|-------------|
| Role | Platform owner or internal administrator. |
| Tech level | Medium to high. Responsible for account governance. |
| Goal | Manage users and assign roles. |
| Frustration | Systems where all users have the same access level. |
| Scenario | Reviews registered users and promotes a trusted user to admin. |

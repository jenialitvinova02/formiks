## Elevator Pitch

Formics is a full-stack web platform for creating, publishing, completing, and analyzing dynamic forms. It is designed for small organizations, educational teams, and internal communities that need more control than a simple public survey tool provides. The product supports roles, public and private access, response management, automatic answer checking, answer analytics, and live activity analytics. Unlike a static form builder prototype, Formics includes a real API, relational database, cloud deployment, containerization, tests, and production-ready configuration.
